# Intern Performance Prediction Using Machine Learning

## Project Overview

This project predicts intern performance using Machine Learning based on attendance, task completion, mentor feedback, learning hours, meetings attended, and deadlines missed.

The project includes data preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation, and prediction using the Random Forest Classifier.

---

## Objectives

- Analyze intern performance
- Visualize internship trends
- Build a Machine Learning prediction model
- Identify key performance factors
- Provide business recommendations

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Dataset Features

- Intern ID
- Attendance (%)
- Tasks Completed
- Average Task Score
- Mentor Feedback
- Meetings Attended
- Learning Hours
- Deadlines Missed
- Performance

---

## Machine Learning Model

- Random Forest Classifier

---

## Results

The Random Forest model predicts intern performance using attendance, mentor feedback, learning hours, task completion, and deadlines missed. The project also includes data visualization and business recommendations for mentors.
